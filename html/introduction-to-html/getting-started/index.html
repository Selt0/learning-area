Starting A Project

  1. Make a directory
    • CMD Line: mkdir repos/<name-of-project>
  2. Change into the directoty and create a file (index.html)
    • CMD Line: cd repos/<name-of-project>
    • CMD Line: touch index.html
  3. Initialize the repository and make a commit

    4. Note: Be sure you're in the project directory

    • CMD Line: git init
    • CMD Line: git add -A && git commit -am "Initialize repo"
  4. Set the remote origin as the main URL and push
    • CMD Line: git remote add origin <repo URL>
    • CMD Line: git push -u origin master
  5. Begin working on the prokect, adding and committing changes
    • (To add new files) CMD Line: git add -A
    • CMD Line: git commit -am "(message)"
  6. Finally, push changes to the repo
    • CMD Line: git push
